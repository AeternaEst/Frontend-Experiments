
Setup:

1. create package.json with -> npm init
2. Install babel locally with -> npm install babel-cli babel-core --save-dev
3. Install all ES6 transpiles with -> npm install babel-preset-es2015 --save-dev
4. update package.json script section with:
	"scripts": {
		"babel": "babel --presets es2015 src/es6.js -o lib/main.bundle.js"
  }
5. run babel with the name you selected in the scripts section with:
	npm run babel
